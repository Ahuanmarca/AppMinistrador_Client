##  Overview

The AppMinistrador App is our capstone project for the Full Stack Developer Bootcamp at EDEM, powered by The Bridge, in Valencia, Spain. Alongside the Cybersecurity team, we have embraced the challenge proposed by the Colegio de Administradores de Fincas de Valencia y Castell√≥n. The project's MVP features a dashboard providing fundamental information on estate management.

##  Getting Started

  

To get started with the project, follow the instructions below.

  

##  Installation

  

1.  **Clone the repository**

```bash

git clone https://github.com/Ahuanmarca/AppMinistrador_Client.git

```
2.  **Navigate to the project directory:**
```bash

cd AppMinistrador_client

```

3.  **Switch to the relevant branch**

```bash

git checkout [your-branch-name]

```

4.  **Install dependencies**

```bash

npm install

```

5.  **Start the application**

```bash

npm run dev

```

This will run the app in the development mode. Access [http://localhost:5173](http://localhost:5173) to view it in the browser.

Make sure you have [Node.js](https://nodejs.org/) installed on your machine before running these commands.
